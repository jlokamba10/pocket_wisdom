# PocketWisdom Feedback Setup (Non-Technical)

## What you need first
- Docker Desktop installed
- Docker Desktop open and running

## First-time setup (about 5-10 minutes)
1. Unzip the folder you received.
2. Open the unzipped folder.
3. Double-click `start-feedback.bat`.
4. Wait until you see `PocketWisdom is ready.`
5. Open your browser and go to `http://localhost:5173`

## Login
- Email: `sysadmin@pocketwisdom.local`
- Password: `Admin123!`

## Optional dashboards
- Grafana URL: `http://localhost:3000`
- Username: `admin`
- Password: `admin`

## Stop the system
1. Go back to the project folder.
2. Double-click `stop-feedback.bat`.

## If something goes wrong
- Make sure Docker Desktop is open.
- If the start window shows an error, close it and run `start-feedback.bat` again.
- If needed, restart Docker Desktop and run `start-feedback.bat` again.
